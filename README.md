_# SmartTrade
A multi-vendor electronic selling E-Commerce web application

---

## Overview

SmartTrade is a full-featured multi-vendor e-commerce platform designed to simplify online electronic product trading. It allows multiple vendors to manage their products, customers to shop online, and admins to oversee transactions and user management efficiently.

---

## Configuration Setup

Before running the project, make sure to create a configuration file named:
and add the following properties:

```properties
db.url=jdbc:mysql://localhost:3306/smarttrade
db.username=your_username_here
db.password=your_password_here
mail.username=your_email_here
mail.password=your_email_password_here
```

## How to Run
Clone the repository:
git clone https://github.com/your-username/SmartTrade.git



Open the project in IntelliJ IDEA or your preferred IDE.
Configure the database:
Create a new MySQL database named smarttrade.
Update the credentials in app.properties as shown above.
Build and run the project using Maven or your IDE's build tool.

Access the application via:
http://localhost:8080/SmartTrade
